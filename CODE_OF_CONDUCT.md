Code of Conduct

Be respectful.
Be constructive.
Assume good intent.

We welcome:
	•	Beginners
	•	Experts
	•	First-time contributors
	•	People asking “dumb” questions (there are none)

We do not welcome:
	•	Personal attacks
	•	Gatekeeping
	•	Hostility
	•	Drama

This project is about building something cool together.

If something feels off, open an issue or reach out.
